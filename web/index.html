KeyPressCounter

A lightweight Windows system tray utility that monitors keyboard and mouse input patterns alongside real-time system performance metrics — privately, with no data leaving your machine.

⬇️ Download Latest Release
⚠️

Windows Desktop Application — KeyPressCounter is a native Windows WinForms app targeting .NET 10.0. It requires Windows 10 x64 or later and cannot be run directly in a browser. Download the release ZIP from GitHub and run MWH.KeyPressCounter.exe on your Windows machine.

Features
⌨️

Input Tracking

Counts total keystrokes and mouse clicks since the last reset. Tracks peak activity rates per minute without recording what you type.

📊

System Performance

Real-time CPU, memory, disk I/O, and network metrics with 60-second rolling history graphs.

🔕

Idle Detection

Automatically detects idle periods and optionally excludes them from activity statistics.

🗂️

Activity Logs

Writes timestamped activity logs every 60 seconds and daily summary logs at midnight.

🔒

Private by Design

No keystrokes are recorded. No data leaves the machine. All storage is local only.

🖥️

System Tray UI

Runs unobtrusively in the system tray. Three-tab stats dashboard opens on double-click.

Getting Started (Windows)
1

Download the latest KeyPressCounter-vX.X.X-win-x64.zip from the Releases page.

2

Install the .NET 10.0 Runtime if not already installed. Verify with dotnet --version.

3

Extract the ZIP to any folder and run MWH.KeyPressCounter.exe. The app appears in the system tray immediately.

4

Double-click the tray icon to open the statistics dashboard, or right-click for the full context menu.

Building from Source
1

Clone the repository: git clone https://github.com/MarkHazleton/KeyPressCounter.git

2

Build: dotnet build MWH.KeyPressCounter.csproj --configuration Release

3

Run: dotnet run --project MWH.KeyPressCounter.csproj (requires Windows)
